Object Relational Mapping for front-ends

Getting Started â†’

Strictly Typed

Enjoy the pleasant development experience brought by the full range of strict typing, and maximize the advantage of TypeScript.

Lightweight and Universal

Focus on and only focus on Object Relational Mapping, no more, no less, easy to integrate into any framework.

Easy to Use

There are not many APIs, invoking just one method is enough to convert the original data into an entity and allow you to enjoy the complete object relations, and the object relations can be updated simply by assigning a new value.

  • Directly resolve the data obtained from the backend to get the entity
    const userData = await axios.get("/users/1/");
const user = em.populate(User, userData);
  • All the entities are instances of their entity classes
    user instanceof User;
  • Values of relation fields are no longer just a primary key or an array of primary keys
    user.profile instanceof Profile;
user.friends instanceof Collection;
user.friends.forEach((friend) => friend instanceof User);
  • Relations are bilateral, either side of the relations can access the other side
    user.profile.owner == user;
user.friends.forEach((friend) => friend.friends.has(user));
