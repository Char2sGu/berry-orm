# 自定义存储

默认地，Berry ORM 会在内部创建IdentityMap来保存一切实体的引用，你需要额外自行管理你的数据。

对于某种实体，如果你想要和 Berry ORM 共用一个存储，可以在@Entity()中设置。

 





@Entity({ map: () => yourMap })
export class User extends BaseEntity<User, "id"> {
  @Field({ type: "primary" })
  id!: number;
}
